Notes about using Hyphenator.js

This site contains some hints about using Hyphenator and uses it itself. So it's a sort of testcase, too. To see in-word line-breaking, change the size of the window.

Version

This text refers to version 5 of the script. Please have a look at http://code.google.com/p/hyphenator/ for further information. Hyphenator can be used on english, german and french webpages.

Use Hyphenator on your webpage

Include the script in the <head>-Block of your webpage:
<script src="http://hyphenator.googlecode.com/svn/trunk/Hyphenator.js" type="text/javascript"> </script>
By including the script the object Hyphenator is automatically generated.

The next step is to run Hyphenator:
<script type="text/javascript">     Hyphenator.run(); </script>

The script searches all elements with the class "hyphenate" (e.g. <p class="hyphenate">) and hyphenates text and child-nodes in this elements. To set a specific language, lang has to be set in at least one parent element.

Use as a Bookmarklet

To hyphenate a foreign site (e.g. before you print it) you may install the following Bookmarklet. It loads the script and hyphenates all elements, so it's slower than the original script.

javascript: if (document.createElement) { void(head=document.getElementsByTagName('head').item(0)); void(script=document.createElement('script')); void(script.src='http://www.mnn.ch/hyph/v5/Hyphenator.js?bm=true'); void(script.type='text/javascript'); void(head.appendChild(script));}
(save this link as a bookmark).

Hyphenation in document with miscellaneous languages

With the attribute lang you can specify the language of an elements text; that way, Hyphenator can load language-specific patterns. As an example, here is a french and english text by J.J. Rousseau:
FR: Je forme une entreprise qui n'eut jamais d'exemple, et dont l'exécution n'aura point d'imitateur. Je veux montrer à mes semblables un homme dans toute la vérité de la nature; et cet homme, ce sera moi.
EN: I have entered upon a performance which is without example, whose accomplishment will have no imitator. I mean to present my fellow-mortals with a man in all the integrity of nature; and this man shall be myself.
